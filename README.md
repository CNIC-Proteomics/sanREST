# sanREST
A RESTful API that describes the web services and the operations for the report files of iSanXoT.

To start it, install Node:
```
npm install
```

And run (on Windows):
```
npm run start.win
```
And run (on Linux):
```
npm run start
```

# RUN SCRIPTS

`` 
npm run restart
npm run restart.win
npm run stopall
npm run restartall
```
